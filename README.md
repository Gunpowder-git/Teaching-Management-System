# Teaching-Management-System
一个由 C++ 教学管理系统迁移而来的 Python 课程作业项目，实现学生成绩管理、绩点计算、学分加权排序、CSV 导出与 matplotlib 可视化。A Python course project migrated from a C++ teaching management system, featuring student grade management, GPA calculation, credit-weighted ranking, CSV export, and matplotlib visualizations.
